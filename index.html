react-appear-in

Make React components appear sometime in the future

Github NPM

Install

npm i react-appear-in
Or
yarn add react-appear-in

react >=16.8.0 is required as peer dependency.

Use

import AppearIn from 'react-appear-in';

const MyAwesomeComponent = () => (
    <AppearIn minutes={1} seconds={20} milliseconds={10}>
        Hello in 1 minute, 20 seconds and 10 milliseconds
    </AppearIn>
);

Props

PropTypeNotes
minutesnumberOptional
secondsnumberOptional
millisecondsnumberOptional
placeholder(time: number) => ReactNodeOptional. time is in milliseconds.
onAppear(time: number) => voidOptional. time is in milliseconds.
